Camelot¶
Camelot¶

Camelot provides components for building applications on top of Python, SQLAlchemy and Qt.
除了Python，SQLAlchemy和Qt之外，Camelot 提供构建应用的组件

It is inspired by the Django admin interface.
它的灵感取决于Django的管理接口

The main resource for information is the website: http://www.python-camelot.com and the mailing list https://groups.google.com/forum/#!forum/project-camelot
如下是主要的信息资源网站: http://www.python-camelot.com ，邮件列表网站： http://groups.google.com/forum/#!forum/project-camelot

GTk¶
GTk¶

PyGTK provides Python bindings for the GTK+ toolkit.
PyGTK提供GTK+工具包的Python绑定

Like the GTK+ library itself, it is currently licensed under the GNU LGPL.
和GTK+库一样，PyGTK当前由GNU LGPL授权

It is worth noting that PyGTK only currently supports the Gtk-2.X API (NOT Gtk-3.0).
PyGTK当前只支持Gtk-2.X的API（Gtk-3.0不支持）

It is currently recommended that PyGTK not be used for new projects and that existing applications be ported from PyGTK to PyGObject.
值得一提的是，PyGTK不适用于新项目，而已经存在的应用不能从PyGTK移植到PyGObject

PyGObject aka (PyGi)¶
Kivy¶

PyGObject provides Python bindings, which gives access to the entire GNOME software platform.
Kivy 是一个Python库，该库用于开发多点触控的媒体应用

It is fully compatible with GTK+ 3.
当代码需要重复利用并且可部署时，它能够实现快速简单的交互设计，并且加速成形

API Reference
Kivy使用Python编写，并且基于OpenGL，除此，它支持不同的输入设备，例如鼠标，双鼠标，WiiMote，WM_TOUCH，HIDtouch和苹果的产品等等

Kivy is written in Python, based on OpenGL and supports different input devices such as: Mouse, Dual Mouse, TUIO, WiiMote, WM_TOUCH, HIDtouch, Apple’s products and so on.
如下是主要的信息资源网站： http://kivy.org

Only available on OS X.
仅在OS X中可用

Don’t pick this if you’re writing a cross-platform application.
如果你正在写一个跨平台的应用，不要选用这个

PyjamasDesktop (pyjs Desktop)¶
PyjamasDesktop (pyjs Desktop)¶

PyjamasDesktop is a port of Pyjamas.
PyjamasDesktop是Pyjamas的端口

PyjamasDesktop is application widget set for desktop and a cross-platform framework.
PyjamasDesktop是桌面应用工具集，并且是跨平台框架

(After release v0.6 PyjamasDesktop is a part of Pyjamas (Pyjs)).
（在发布的v0.6版本之后，PyjamasDesktop是Pyjamas (Pyjs)的一部分）

Briefly, it allows the exact same Python web application source code to be executed as a standalone desktop application.
简而言之，它允许完全一样的Python网页应用资源代码能够如独立的桌面应用执行

Python Wiki for PyjamasDesktop.
PyjamasDesktop的Python维基百科.

The main website;
主要的网址: pyjs Desktop.

Qt¶
Qt¶

Qt is a cross-platform application framework that is widely used for developing software with a GUI but can also be used for non-GUI applications.
Qt 是跨平台应用框架，它被广泛用于借GUI开发软件，但是也可用于非GUI应用

Tk¶
Tk¶

Tkinter is a thin object-oriented layer on top of Tcl/Tk.
Tkinter是Tcl/Tk上的面向对象层

It has the advantage of being included with the Python standard library, making it the most convenient and compatible toolkit to program with.
 它的优势是包括Python标准库，能够使编程更加方便，兼容性更强

Both Tk and Tkinter are available on most Unix platforms, as well as on Windows and Macintosh systems.
不管是Tk还是Tkinter，在大多数Unix平台，以及Windows和Macintosh系统都可用

Starting with the 8.0 release, Tk offers native look and feel on all platforms.
从8.0发布版本开始，Tk在所有平台上使本身的样式和感觉更赞

There’s a good multi-language Tk tutorial with Python examples at TkDocs.
在 TkDocs 中有一个非常好的多语言Tk教程，所有例子使用Python

There’s more information available on the Python Wiki.
更多信息可以看 Python 维基百科.

wxPython¶
wxPython¶

wxPython is a GUI toolkit for the Python programming language.
wxPython是Python语言编写的GUI工具包

It allows Python programmers to create programs with a robust, highly functional graphical user interface, simply and easily.
Python编写人员能够使简单容易地使用健壮，高功能的图形用户接口编程

It is implemented as a Python extension module (native code) that wraps the popular wxWidgets cross platform GUI library, which is written in C++.
把流行的wxWidgets包在跨平台GUI库中，从而作为Python的扩展模块，这用C++编写

Install (Stable) wxPython go to http://www.wxpython.org/download.php#stable and download the appropriate package for your OS.
安装（稳定版）wxPython可以去如下网址 http://www.wxpython.org/download.php#stable ，并且下载适用于当前操作系统的安装包
