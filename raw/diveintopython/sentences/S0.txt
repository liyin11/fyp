1.2. Python on Windows
1.2. Windows 上的 Python

On Windows, you have a couple choices for installing Python.
在 Windows 上，安装 Python 有两种选择

ActiveState makes a Windows installer for Python called ActivePython, which includes a complete version of Python, an IDE with a Python-aware code editor, plus some Windows extensions for Python that allow complete access to Windows-specific services, APIs, and the Windows Registry.
ActiveState 制作的 ActivePython 是专门针对 Windows 的 Python 套件，它包含了一个完整的 Python 发布、一个适用于 Python 编程的 IDE 以及一些 Python 的 Windows 扩展，提供了全部的访问 Windows APIs 的服务，以及 Windows 注册表的注册信息

ActivePython is freely downloadable, although it is not open source.
虽然 ActivePython 不是开源软件，但它可以自由下载

It is the IDE I used to learn Python, and I recommend you try it unless you have a specific reason not to.
ActivePython 是我学习 Python 时使用过的 IDE

One such reason might be that ActiveState is generally several months behind in updating their ActivePython installer when new version of Python are released.
除非有别的原因，我建议您使用它

If you absolutely need the latest version of Python and ActivePython is still a version behind as you read this, you'll want to use the second option for installing Python on Windows.
可能的一个原因是：ActiveState 通常要在新的 Python 版本发布几个月以后才更新它的安装程序

The second option is the “official” Python installer, distributed by the people who develop Python itself.
第二种选择是使用由 Python 发布的 “官方” Python 安装程序

It is freely downloadable and open source, and it is always current with the latest version of Python.
她是可自由下载的开源软件，并且您总是可以获得当前 Python 的最新版本

Here is the procedure for installing ActivePython:
下面描述 ActivePython 的安装过程：

Download ActivePython from http://www.activestate.com/Products/ActivePython/.
从 http://www.activestate.com/Products/ActivePython/ 下载 ActivePython 

If you are using Windows 95, Windows 98, or Windows ME, you will also need to download and install Windows Installer 2.0 before installing ActivePython.
如果您正在使用 Windows 95、Windows 98 或 Windows ME，还需要在安装 ActivePython 之前下载并安装Windows Installer 2.0 

Double-click the installer, ActivePython-2.2.2-224-win32-ix86.msi.
双击安装程序 ActivePython-2.2.2-224-win32-ix86.msi

Step through the installer program.
按照安装程序的提示信息一步步地执行

If space is tight, you can do a custom installation and deselect the documentation, but I don't recommend this unless you absolutely can't spare the 14MB.
如果磁盘空间不足，您可以执行定制安装，不选文档，但是笔者不建议您这样做，除非您实在是挤不出14M空间来

After the installation is complete, close the installer and choose Start->Programs->ActiveState ActivePython 2.2->PythonWin IDE.
在安装完后之后，关闭安装程序，打开 开始->程序->ActiveState ActivePython 2.2->PythonWin IDE

You'll see something like the following:
您将看到类似如下的信息：

Download the latest Python Windows installer by going to http://www.python.org/ftp/python/ and selecting the highest version number listed, then downloading the .exe installer.
从 http://www.python.org/ftp/python/ 选择最新的 Python Windows 安装程序，下载 .exe 安装文件

Double-click the installer, Python-2.xxx.yyy.exe.
双击安装程序 Python-2.xxx.yyy.exe

The name will depend on the version of Python available when you read this.
文件名依赖于您所下载的 Python 安装程序文件

Step through the installer program.
按照安装程序的提示信息一步步地执行

If disk space is tight, you can deselect the HTMLHelp file, the utility scripts (Tools/), and/or the test suite (Lib/test/).
如果磁盘空间不足，可以取消 HTMLHelp 文件、实用脚本 (Tools/)、和/或测试套件 (Lib/test/)

If you do not have administrative rights on your machine, you can select Advanced Options, then choose Non-Admin Install.
如果您没有机器的管理员权限，您可以选择 Advanced Options，然后选择 Non-Admin Install

This just affects where Registry entries and Start menu shortcuts are created.
这只会对登记注册表和开始菜单中创建的快捷方式有影响

After the installation is complete, close the installer and select Start->Programs->Python 2.3->IDLE (Python GUI).
在安装完成之后，关闭安装程序，打开 开始->程序->Python 2.3->IDLE (Python GUI)

You'll see something like the following:
您将看到类似如下的信息：
